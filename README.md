# Mi Lonchera Saludable 🍱

Un juego web interactivo diseñado para niños de **Preescolar (Fase 2)** alineado con los principios de la **Nueva Escuela Mexicana (NEM) 2022**. El propósito del juego es enseñar a los niños a diseñar loncheras saludables, balanceadas y reconocer la importancia del agua simple potable, además de identificar alimentos que pueden poner en riesgo la salud en caso de consumo excesivo.

## 🎯 Alineación Curricular (NEM 2022)

* **Campo Formativo:** De lo Humano y lo Comunitario.
* **Eje Articulador:** Vida Saludable.
* **Contenido:** *Consumo de alimentos y bebidas que benefician la salud, de acuerdo con los contextos culturales, en el favorecimiento de una vida saludable.*
* **Procesos de Desarrollo de Aprendizaje (PDA):**
  * **1er Grado:** Compara de forma guiada alimentos saludables de los que no lo son tanto.
  * **2do Grado:** Distingue alimentos y bebidas saludables de aquellos que ponen en riesgo la salud.
  * **3er Grado:** Obtiene información acerca de los beneficios del consumo de alimentos saludables y agua simple potable para el diseño de menús y loncheras balanceadas.

---

## 🚀 Características del Juego

1. **Plato del Bien Comer interactivo:** Clasificación visual de alimentos (Frutas/Verduras, Cereales, Proteínas/Lácteos) y Agua simple.
2. **Síntesis de Voz (Accesibilidad):** Ideal para niños pequeños que aún no saben leer. El juego pronuncia los nombres de los alimentos, explica sus beneficios y guía al jugador por voz.
3. **Indicador de Balance:** Monitoreo dinámico del progreso de la lonchera en tiempo real.
4. **Filtros Pedagógicos:** Filtros rápidos para que el docente o alumno exploren las categorías alimenticias.
5. **Sección de Alimentos Ocasionales/Chatarra:** Permite a los niños aprender a identificar y evitar alimentos altos en azúcares y grasas, mostrando advertencias amigables.

## 👥 Cómo colaborar en este proyecto (GitHub)

¡Este es un proyecto colaborativo! Si quieres mejorar el juego, agregar nuevos alimentos o cambiar el diseño, sigue estos pasos:

### 1. Clonar el repositorio
Si quieres trabajar en tu computadora localmente:
```bash
git clone <URL_DEL_REPOSITORIO>
cd COLABORADORES
```

### 2. Crear una rama (Branch) para tus cambios
Nunca hagas cambios directamente en la rama `main` (principal). Crea una rama descriptiva:
```bash
git checkout -b agregar-nuevas-frutas
```

### 3. Hacer tus cambios y subirlos (Push)
Una vez que edites el código:
```bash
git add .
git commit -m "Agrega fresas y sandías a la categoría de frutas"
git push origin agregar-nuevas-frutas
```

### 4. Abrir un Pull Request (PR)
Ve a la página de tu repositorio en GitHub y haz clic en **New Pull Request** para solicitar que tus cambios se fusionen a la rama principal. ¡Ahí los revisaremos juntos!

## 🌐 Despliegue con GitHub Pages

Para que cualquier persona (niños, padres y profesores) pueda jugar gratis desde el navegador:
1. Ve a tu repositorio en GitHub.
2. Ve a **Settings** (Configuración) > **Pages** en el menú lateral.
3. En la sección *Build and deployment*, bajo **Source**, selecciona **Deploy from a branch**.
4. Selecciona la rama `main` y la carpeta `/ (root)`.
5. Haz clic en **Save** (Guardar).
6. En unos minutos, GitHub te dará un enlace público (ej. `https://tu-usuario.github.io/COLABORADORES/`) para que juegues y lo compartas con tus alumnos.
